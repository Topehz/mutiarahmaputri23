CARA KIRIM WEBSITE ULANG TAHUN INI

ISI FOLDER INI SUDAH SIAP UPLOAD:
- index.html
- vercel.json
- foto .jpeg
- lagu .mp3
- video .mp4

CARA PALING MUDAH PAKAI VERCEL:
1. Buka https://vercel.com
2. Login atau daftar akun.
3. Klik Add New > Project.
4. Upload folder ini, atau upload lewat GitHub kalau kamu sudah biasa.
5. Framework Preset pilih Other.
6. Build Command kosongkan.
7. Output Directory kosongkan.
8. Klik Deploy.
9. Setelah selesai, kamu akan dapat link gratis berakhiran .vercel.app.
10. Coba ubah nama project/domain gratisnya menjadi mutiarahmaputri23 supaya link-nya menjadi:
    https://mutiarahmaputri23.vercel.app

OPSI DOMAIN GRATIS LAIN:
- Vercel: mutiarahmaputri23.vercel.app
- Netlify: mutiarahmaputri23.netlify.app
- Cloudflare Pages: mutiarahmaputri23.pages.dev

Domain gratis ini tergantung ketersediaan nama. Kalau sudah dipakai orang lain, tambahkan angka atau kata, contoh:
- mutiarahmaputri23-birthday.vercel.app
- birthday-mutia23.vercel.app
- for-mutia-23.vercel.app

CARA PAKAI DOMAIN MUTIARAHMAPUTRI23.COM:
1. Beli dulu domain mutiarahmaputri23.com di penyedia domain.
2. Di Vercel, buka project yang sudah di-deploy.
3. Buka Settings > Domains.
4. Masukkan mutiarahmaputri23.com.
5. Ikuti DNS record yang diminta Vercel.
6. Tunggu propagasi DNS, biasanya beberapa menit sampai beberapa jam.

CATATAN:
- Fitur tiup lilin pakai mic sebaiknya dibuka lewat link https dari Vercel.
- Kalau nama file foto/lagu/video diganti, samakan juga namanya di DATA_ULANG_TAHUN di index.html.
